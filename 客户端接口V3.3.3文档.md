# 亿驾 v3.3.3+ 客户端交互接口文档

# 一、 背景说明

本接口文档服务于新版亿驾客户端(`v3.3.3`及以上版本)，与 `v3.0` 文档的主要区别在于有的接口返回更多数据，其他与V3.0一致，当客户端同时可调用V3.0的接口。由于本版本接口大致于V3.0相同，此文档只说明特有的接口。如需查询其他接口可参考V3.0接口文档。

# 二、接口列表及示例说明
